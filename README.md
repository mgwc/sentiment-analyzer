Analyzes sentiment (i.e. degree of positivity or negativity) of a sentence based on its words. Words are assigned sentiment values based on their use in a corpus of IMDB reviews (see reviews.txt). Sentiment values range from -2 (most negative) to 2 (most positive).

To run:
Compile the program using the command "make all".
Enter "./Main reviews.txt" in the command line.
Enter a sentence as directed by the program.
Type "quit" to quit.
